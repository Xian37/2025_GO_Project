# Multifunction Chatroom
This is a program that can let you set up your own chatroom server on your local net.

## Main Features
- Custom profile picture and nickname.
- Text chat and voice chat availible at same time.
- Easy emoji reactions to any text message.
- Create personal rooms with password
- Fun mini games to play with people in the same room
    - Guess number
    - Draw and Guess
    - Riddle

## How to Use
1. Install Golang.
2. Download the zip and unzip it.
3. Go in the chatroom folder and run `go run main.go`.
4. Open `http://localhost:8080/` in the browser and enjoy your time.
